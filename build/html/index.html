class ispi.SpeedTest¶

Instances of this class are used to test download speeds using speedtest-cli.

create_db()¶

Create the sqlite database at the path identified by _get_full_db_path()_

get_full_db_path()¶

Returns the absolute path to the database, including the database’s name.

get_speeds(speedtest_results)¶

Input: results from speedtest as a string, Output: A Dict with keys “download_speed” and “upload_speed”

result_code()¶

Returns a code for success or failure.

run_speedtest()¶

Runs the main method of speedtest-cli.py, captures and returns the stdout output in a StringIO object.

save_results(result_dict)¶
Input: Result dict from get_speeds with keys
‘timestamp’ ‘download_speed’, ‘upload_speed’
